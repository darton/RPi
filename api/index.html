RPiMS API

To download current data in JSON format, use the following address:

http://ipaddress/api/data

In order to obtain data only for a selected sensor or a group of sensors, the address should be filled in with a parameter, e.g .:

RPiMS Setting

http://ipaddress/api/data/settings

RPiMS Sensors

http://ipaddress/api/data/sensors

BME280 sensor

http://ipaddress/api/data/sensors/bme280

Weather Station sensors

http://ipaddress/api/data/weather_station
